EventsX - Event Planning Web Application

Project Overview
EventsX is a comprehensive, responsive web application designed to streamline event planning and management. Built as our final project for CSE211 Web Programming, this platform provides users with intuitive tools to browse venues, calculate budgets, and register for events—all within a seamless, modern interface.
Key Features
-Interactive Venue Slider  – Browse 12+ event spaces with smooth horizontal navigation
-Smart Budget Calculator – Plan event finances with real-time updates and visual progress indicators
-Form Validation System – Register with immediate feedback and error prevention
- Multi-Channel Contact – Connect via forms, direct contact info, FAQ, and embedded Google Maps
- Responsive Design – Fully functional across mobile, tablet, and desktop devices
